# Lumina E-Commerce

Lumina is a premium, minimalist e-commerce application platform designed with a focus on modern aesthetics and streamlined user experience. It features a complete customer portal and a powerful administration dashboard.

## 🚀 Features

### Customer Experience
- **Premium Interface**: A minimalist, high-contrast design using glassmorphism and subtle animations.
- **Product Discovery**: Browse collections with smooth transitions and detailed product views.
- **Responsive Design**: Optimized for everything from mobile devices to ultra-wide displays.

### Admin Dashboard
- **Real-time Analytics**: Visual data representations using Recharts for sales, revenue, and customer trends.
- **Collapsible Sidebar**: A flexible management menu that can be collapsed for a cleaner workspace.
- **Inventory Management**: Full control over products, orders, and customer data.
- **Order Tracking**: Comprehensive registry for managing customer transactions and fulfillment statuses.

## 🛠️ Tech Stack

- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion (via `motion/react`)
- **Charts**: Recharts
- **Icons**: Lucide React
- **Routing**: React Router 7
- **Bundler**: Vite 6

## 📦 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Project Structure

- `src/components/`: Reusable UI components (Navbar, Footer, Admin layouts).
- `src/pages/`: Main application views (Home, Shop, Admin Dashboard, etc.).
- `src/lib/`: Utility functions and shared helpers.
- `src/routes.tsx`: Centralized routing configuration.
